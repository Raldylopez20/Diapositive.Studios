# Diapositive Studio - Portafolio Interactivo 🚀

Landing page y portafolio interactivo premium para **Diapositive Studio**, agencia de marketing digital, diseño y fotografía profesional.

## 🔗 Enlace del Proyecto (GitHub Pages)
Puedes ver el sitio web en vivo aquí:
👉 **[https://raldylopez20.github.io/Diapositive.Studios/](https://raldylopez20.github.io/Diapositive.Studios/)**

---

## ✨ Características Principales
* **Diseño Premium y Moderno:** Animaciones interactivas de entrada, modo oscuro refinado y transiciones fluidas.
* **Portafolio Dinámico:** Galería interactiva con modal (lightbox) para visualizar los proyectos a pantalla completa.
* **Carga Ultra Rápida:** Optimización completa de recursos e imágenes convertidas al formato de última generación `.avif` (máxima calidad visual con bajo peso).
* **Lógica Inteligente en Galerías:** Ajuste automático en JavaScript que oculta secciones de descripción vacías en el modal de forma limpia.
* **Diseño Responsive:** Totalmente adaptado para dispositivos móviles, tablets y computadoras de escritorio.

---

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Estilos vanilla premium, variables personalizadas y animaciones.
* **JavaScript (ES6):** Funcionalidad interactiva del carrusel, modal y controles dinámicos.
* **AOS (Animate On Scroll):** Animaciones al hacer scroll.
* **FontAwesome:** Iconografía moderna.
